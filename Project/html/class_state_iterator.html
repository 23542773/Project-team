Detailed Description

A concrete iterator designed to skip over elements and only yield Plant objects whose internal state pointer matches a predefined target PlantState.

This iterator provides a filtered view of the Greenhouse collection, focused on plants sharing a specific lifecycle state (e.g., only "MatureState" plants).
