Detailed Description

Data structure representing a message between colleagues.

This structure encapsulates all information about a message sent through the mediator system. Messages are created by the ChatMediator and delivered to both sender and recipient for conversation history tracking.

Each message has a unique ID, sender and recipient identifiers, the message content, and a timestamp for chronological ordering.
