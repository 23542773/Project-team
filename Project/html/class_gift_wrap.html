Detailed Description

Concrete Decorator adding gift wrapping service to sale items.

This decorator enhances a plant sale item by adding professional gift wrapping, making it suitable for presents and special occasions. The gift wrap includes decorative paper, ribbon, and presentation suitable for gifting.

Adds a fixed service charge and updates the description to indicate gift wrapping. Commonly combined with MessageCard decorator for complete gift presentation.
