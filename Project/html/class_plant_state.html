Detailed Description

Abstract State class in the State pattern representing different plant growth states.

This class defines the interface for all concrete plant states. Each state encapsulates the behavior associated with a particular stage of a plant's lifecycle and determines transitions to other states based on plant health, moisture, insecticide levels, and age.
