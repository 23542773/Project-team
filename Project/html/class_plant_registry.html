Detailed Description

Manages a collection of unique prototype Plant objects, keyed by their SKU.

This class serves as the Prototype Registry, allowing clients (like the Greenhouse) to retrieve and clone new Plant instances without needing to know the complex creation logic of the concrete Plant classes.
