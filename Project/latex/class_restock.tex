\doxysubsection{Detailed Description}
A concrete \doxylink{class_command}{Command} that encapsulates the request to receive a shipment (restock) of new plants into the \doxylink{class_greenhouse}{Greenhouse}. 

This command supports undo functionality by tracking the unique IDs of the plants that were added during execution.
