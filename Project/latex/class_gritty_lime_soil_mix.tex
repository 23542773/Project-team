\doxysubsection{Detailed Description}
A concrete implementation of the \doxylink{class_soil_mix}{Soil\+Mix} abstract class, representing a specific type of soil. 

This class is designed to function as a {\bfseries{Prototype}} within a registry, allowing new instances of this soil type to be created via cloning without knowing the details of its construction.
