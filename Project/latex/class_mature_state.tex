\doxysubsection{Detailed Description}
Concrete State class representing a fully grown plant. 

This state handles the logic for a mature plant with higher resource demands. It can transition to \doxylink{class_wilting_state}{Wilting\+State} or \doxylink{class_dead_state}{Dead\+State} if care is inadequate.
