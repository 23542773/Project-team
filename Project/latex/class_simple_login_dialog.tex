\doxysubsection{Detailed Description}
Dialog for user authentication and role selection. 

Provides a simple login interface where users enter their ID and select their role (\doxylink{class_customer}{Customer} or \doxylink{class_staff}{Staff}). The dialog validates credentials through the \doxylink{class_nursery_facade}{Nursery\+Facade} before allowing access.
