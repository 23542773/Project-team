\doxysubsection{Detailed Description}
Manages the collection of all living \doxylink{class_plant}{Plant} objects, tracking them by ID and SKU. 

This class is responsible for\+:
\begin{DoxyEnumerate}
\item Managing the lifecycle of plants (via {\ttfamily \doxylink{class_greenhouse_ae99611fb6ed3e9d6240378b88ff3c0f8}{tick\+All}}).
\item Handling new plant shipments.
\item Acting as the {\bfseries{Subject}} ({\ttfamily \doxylink{class_service_subject}{Service\+Subject}}) to notify observers of \doxylink{class_plant}{Plant} events (e.\+g., Matured).
\item Providing {\bfseries{\doxylink{class_iterator}{Iterator}}} factory methods for structured traversal of plants. 
\end{DoxyEnumerate}
