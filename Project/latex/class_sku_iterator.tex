\doxysubsection{Detailed Description}
A concrete iterator designed to skip over elements and only yield \doxylink{class_plant}{Plant} objects whose species Stock Keeping Unit (SKU) matches a predefined target. 

This iterator hides the complexity of filtering the underlying collection from the client.
