\doxysubsection{Detailed Description}
Represents a single plant entity in the greenhouse system. 

A \doxylink{class_plant}{Plant} instance combines shared intrinsic species data (via Flyweight) with its own extrinsic state (moisture, health, etc.). It reacts to external care actions using Strategy and updates its growth phase using the State pattern. The Prototype pattern supports efficient cloning of plant objects for reuse or replication.
