\doxysubsection{Detailed Description}
Concrete State class representing a plant in its primary growth stage. 

This state handles the logic for a healthy plant actively growing. It manages resource consumption and health adjustments, and transitions to \doxylink{class_mature_state}{Mature\+State}, \doxylink{class_wilting_state}{Wilting\+State}, or \doxylink{class_dead_state}{Dead\+State} based on conditions.
