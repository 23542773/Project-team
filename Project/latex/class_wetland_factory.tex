\doxysubsection{Detailed Description}
Concrete factory for wetland plant kits. 

Implements the \doxylink{class_plant_kit_factory}{Plant\+Kit\+Factory} interface to create wetland-\/specific pots and soil mixes, and provides a wetland-\/adapted \doxylink{class_care_strategy}{Care\+Strategy}.
