\doxysubsection{Detailed Description}
Manages all staff members, tracking their assignments and facilitating load balancing. 

This class provides a centralized mechanism for staff-\/related operations, including adding staff, finding the least busy staff member, and managing order assignments. It also holds a reference to the \doxylink{class_messaging_mediator}{Messaging\+Mediator} for staff communication setup.
