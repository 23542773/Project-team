\doxysubsection{Detailed Description}
A composite \doxylink{class_command}{Command} that can execute a list of contained \doxylink{class_command}{Command} objects sequentially. 

This pattern allows clients to treat individual Commands and collections of Commands uniformly. It ensures that execution and undo operations are applied correctly to all sub-\/commands.
