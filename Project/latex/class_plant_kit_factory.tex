\doxysubsection{Detailed Description}
Abstract factory for creating plants and their associated components. 

Provides the interface for creating \doxylink{class_plant}{Plant} objects along with environment-\/specific \doxylink{class_pot}{Pot}, \doxylink{class_soil_mix}{Soil\+Mix}, and \doxylink{class_care_strategy}{Care\+Strategy}. Uses the Factory Method pattern.
