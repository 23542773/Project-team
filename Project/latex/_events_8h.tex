\doxysubsection{Detailed Description}
Event data structures for Observer. 

\begin{DoxyAuthor}{Author}
Damian Moustakis (Doxygen comments) 
\end{DoxyAuthor}
\begin{DoxyDate}{Date}
2025-\/11-\/01
\end{DoxyDate}
The system supports three main event categories\+:
\begin{DoxyItemize}
\item \doxylink{class_plant}{Plant} Events\+: Lifecycle changes in individual plants (wilted, matured, died)
\item Stock Events\+: \doxylink{struct_inventory}{Inventory} level changes (reserved, released, sold, low)
\item \doxylink{struct_order}{Order} Events\+: \doxylink{struct_order}{Order} lifecycle changes (created, assigned, completed, cancelled) 
\end{DoxyItemize}
