\doxysubsection{Detailed Description}
Concrete Flyweight class representing shared intrinsic state of plant species. 

This class implements the Flyweight pattern by storing intrinsic state (data that can be shared across multiple plant instances). All data members represent characteristics that are common to all plants of the same species, making them ideal for sharing to reduce memory consumption.
