\doxysubsection{Detailed Description}
Flyweight Factory that manages and provides access to \doxylink{class_plant_flyweight}{Plant\+Flyweight} objects. 

This class implements the Flyweight Factory pattern, maintaining a pool of shared flyweight objects to minimize memory usage. It ensures that flyweights are shared properly by managing their lifecycle and providing access through unique SKU identifiers.
