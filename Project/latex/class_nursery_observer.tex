\doxysubsection{Detailed Description}
Abstract Base class for all Observers in the Nursery system. 

Defines the interface for handling different event types\+:
\begin{DoxyItemize}
\item \doxylink{class_plant}{Plant} Events\+: Pure virtual method to enforce implementation in derived classes
\item Stock Events\+: Default empty implementation, can be overridden
\item \doxylink{struct_order}{Order} Events\+: Default empty implementation, can be overridden 
\end{DoxyItemize}
