\doxysubsection{Detailed Description}
A concrete iterator designed to skip over elements and only yield \doxylink{class_plant}{Plant} objects whose internal state pointer matches a predefined target \doxylink{class_plant_state}{Plant\+State}. 

This iterator provides a filtered view of the \doxylink{class_greenhouse}{Greenhouse} collection, focused on plants sharing a specific lifecycle state (e.\+g., only "{}\+Mature\+State"{} plants).
