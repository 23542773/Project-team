\doxysubsection{Detailed Description}
Manages a collection of unique prototype \doxylink{class_plant}{Plant} objects, keyed by their SKU. 

This class serves as the {\bfseries{Prototype Registry}}, allowing clients (like the \doxylink{class_greenhouse}{Greenhouse}) to retrieve and clone new \doxylink{class_plant}{Plant} instances without needing to know the complex creation logic of the concrete \doxylink{class_plant}{Plant} classes.
