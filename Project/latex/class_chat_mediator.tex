\doxysubsection{Detailed Description}
Concrete Mediator implementing communication rules between \doxylink{class_staff}{Staff} and Customers. 

This class implements the Mediator pattern by managing all messaging between colleagues in the nursery system. It enforces business rules about who can communicate with whom based on staff roles and user types.

Communication Rules\+:
\begin{DoxyItemize}
\item Customers can only message Sales \doxylink{class_staff}{Staff}
\item Sales \doxylink{class_staff}{Staff} can message Customers, other Sales \doxylink{class_staff}{Staff}, and \doxylink{struct_inventory}{Inventory} \doxylink{class_staff}{Staff}
\item \doxylink{class_plant}{Plant} Care \doxylink{class_staff}{Staff} can message \doxylink{struct_inventory}{Inventory} \doxylink{class_staff}{Staff}
\item \doxylink{struct_inventory}{Inventory} \doxylink{class_staff}{Staff} can message \doxylink{class_plant}{Plant} Care and Sales \doxylink{class_staff}{Staff}
\end{DoxyItemize}

The mediator maintains a registry of all colleagues and routes messages between them while enforcing these constraints, preventing unauthorized communication.
