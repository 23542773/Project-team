\doxysubsection{Detailed Description}
Abstract base class representing the Flyweight interface in the Flyweight design pattern. 

This class defines the interface that all concrete flyweights must implement. It contains methods to access intrinsic state (shared data) that is common across multiple plant instances of the same species.
